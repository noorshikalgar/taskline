# 31 — Design Style Recipes

Purpose: give agents tasteful starting points without copying any brand.

## Recipe: Clean SaaS

```txt
Theme: light, neutral surfaces
Font: modern sans
Radius: 12px
Borders: subtle
Shadows: minimal
Spacing: standard/spacious
Color: one primary accent, sparse status colors
Best for: business tools, dashboards, project apps
```

## Recipe: Developer Console

```txt
Theme: dark or light-neutral
Font: sans + mono for technical data
Radius: 6–10px
Borders: visible but low contrast
Spacing: compact/standard
Color: muted accent, strong focus states
Best for: logs, agents, infra, APIs, CLIs with UI
```

## Recipe: AI Workspace

```txt
Theme: calm neutral, light or dark
Font: readable sans
Radius: 14–18px
Surfaces: cards/panels for threads, outputs, context
Motion: subtle streaming/progress states
Color: avoid magical gradients as main UI; reserve for accent
Best for: chat, agent workflows, automation tools
```

## Recipe: Premium Consumer

```txt
Theme: spacious, high contrast, polished imagery
Font: refined sans/display for marketing only
Radius: 18–24px
Shadows: soft
Spacing: spacious
Color: brand-led but restrained
Best for: mobile comparisons, creator tools, commerce
```

## Recipe: Enterprise Admin

```txt
Theme: light, high clarity
Font: system sans
Radius: 4–8px
Borders: strong enough for dense data
Spacing: compact/standard
Color: blue/neutral default, semantic status
Best for: CRMs, ERPs, security, compliance, operations
```

## Recipe: Editorial Docs

```txt
Theme: high readability
Font: sans for UI, serif optional for long editorial content
Content width: 720–860px
Spacing: generous vertical rhythm
Color: minimal
Best for: docs, learning sites, blogs, guides
```

## Style selection rules

Choose style from product need, not personal taste:

- Frequent daily use → calm, predictable, low visual fatigue.
- Marketing → stronger brand expression.
- Data-heavy → compact, clear hierarchy.
- AI product → transparency, control, confidence.
- Mobile-first → touch comfort and motion clarity.
